# hacktv-gui - a GUI frontend application for hacktv

Please note that this project is now discontinued and has been replaced with a Java rewrite at https://github.com/steeviebops/jhacktv-gui

hacktv-gui is, as the name suggests, a GUI frontend application for hacktv. - https://github.com/fsphil/hacktv/ or https://github.com/captainjack64/hacktv/. It allows you to run hacktv if you have no command line experience. You can also save your settings to a file and reload them later.

This is written in Visual Basic 6.0. It works on virtually any version of Windows (although hacktv itself requires a 64-bit version of Windows Vista or later) and also on Wine with the VB6 runtimes installed. You can download them here - https://www.microsoft.com/en-ie/download/details.aspx?id=24417

I've released this as public domain, feel free to do as you please.
